1. Na co pozwala styl lokalny, który element będzie formatowany z użyciem stylu lokalnego?

Styl lokalny pozwala na nadanie formatowania tylko dla konkretnego elementu HTML poprzez użycie atrybutu style wewnątrz znacznika. Formatowany będzie tylko ten jeden element, w którym umieszczono styl.

2. Składnia teoretyczna stylu lokalnego

<p style="color: red;">Przykładowy tekst</p>

3. Co może być selektorem?

Selektorem w CSS może być:

  • Nazwa znacznika (np. p, h1)
  • Identyfikator elementu (np. #moje-id)
  • Klasa elementu (np. .moja-klasa)
  • Atrybut (np. [type="text"])
  • Pseudoklasa (np. :hover, :first-child)
  • Pseudoelement (np. ::before, ::after)

4. Co wpisujemy jako "cechę"? Jakie jest znaczenie średników?

Cechą jest właściwość CSS, którą chcemy zmienić, np. color (kolor), font-size (rozmiar czcionki). Średniki oddzielają poszczególne deklaracje w regule CSS i są wymagane, aby przeglądarka poprawnie interpretowała kod.

5. Co wpisujemy jako "wartość" (w deklaracji stylu)?

Wartość to konkretna definicja ustawienia dla danej cechy, np. dla color wartością może być red, a dla font-size wartością może być 16px.
